GALLERY

A set of client-side functionality for fetching the gallery images.


Banner, Popup and Background Images


getImages(type) - It'll return all types of images if type is not passed.

It's recommended not to use any client side filtering. Always call methods for filtered images. e.g. getImages('web_banner').

Available types:

  1. web_banner - Images tagged for banners in the website.
  2. mobile_web_banner - Images tagged for banners in the mobile-website.
  3. web_bg_img - Images tagged for background image of the website landing page.
  4. app_bg_img - Images tagged for background image of the mobile-website landing page.
  5. web_popup - Images tagged for Popups.
