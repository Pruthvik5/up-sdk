PAYMENT GATEWAY(web)

A set of client-side functionality for getting integrated with PG details.


Razorpay / Stripe integration:

Just drop the PG Library in your html.


Razorpay:

<script src="https://checkout.razorpay.com/v1/checkout.js"></script>

Stripe:

<script src="https://js.stripe.com/v2/"></script>

To invoke PG, just call .placeOrder(param) with {payment_option:"payment_gateway"} in param.

param is

{ "phone": <phone no>, "payment_option": "payment_gateway", "delivery_datetime": <time in millisecond>, "instructions": "" }

This method calls PG's web-sdk (Not supported in React Native platform) internally. If you want to override Razorpay's default config, call UP_SDK.setPGConfig(<razorpay|stripe>, {options}) and pass option object

Example:

{ "currency": "INR", "name": "PIPERCAFE", "description": "Order payment", "image": "https://thepipercafe.com/images/logo.png?v=1.0", "theme": { "color": "#000" } }

Note: You must have to configure payment gateway in Quint (urbanpiper configuration platform).
